# noAutofocus (since v10.0.0)

> This rule is recommended by Rome.

Avoid the `autoFocus` attribute

## Examples

### Invalid

```jsx
<input autoFocus />
```


```jsx

```

```jsx

<MyComponent autoFocus={true} />
```
