# noRedeclaration (since v12.0.0)

Eliminate variables that have multiple declarations in the same scope.

## Examples

### Invalid

```jsx
var a = 3;
var a = 10;
```

### Valid

```jsx
var a = 3;
a = 10;
```

```ts
class Foo {
    bar(a: A);
    bar(a: A, b: B);
    bar(a: A, b: B) {}
}
```
