# noArguments (since v0.7.0)

Disallow the use of `arguments`

## Examples

### Invalid

```jsx
function f() {
   console.log(arguments);
}
```

### Valid

```js
function f() {
    let arguments = 1;
    console.log(arguments);
}
```
