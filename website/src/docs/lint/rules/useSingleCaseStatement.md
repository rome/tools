# useSingleCaseStatement

Enforces case clauses have a single statement, emits a quick fix wrapping
the statements in a block

## Examples

### Invalid

```jsx
switch (foo) {
    case true:
    case false:
        let foo = '';
        foo;
}
```

### Valid

```jsx
switch (foo) {
    case true:
    case false: {
        let foo = '';
        foo;
    }
}
```
